## Path to completion

2. Restore HomeAgent container onto the test VM using Docker Toolbox (since it's not a native Hyper-V container).
3. Run `docker-compose pull` on the test VM to verify container dependencies are resolved.
4. Verify container logs for potential issues by running `docker logs --tail 100 <container_name>`.
5. Use a secondary device to access the test VM via SSH/RDP and verify functionality of restored containers and data, including network connectivity and storage mounts.
6. Manually unpack Hyper Backup archives from Synology NAS onto the test VM (using rsync or similar tool) for validation purposes.
7. Validate the restored data by comparing it with production data using a diff tool (e.g., Beyond Compare).
